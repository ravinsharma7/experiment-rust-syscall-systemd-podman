# experiment-rust-syscall-systemd-podman
Experimenting what library is used for calling syscall, systemd api, and podman api. General template(not related to web) for me to use for other related rust project to work the os or low level primitives. Then use it as library in golang.

# Calling rust lib from golang
[rust-plus-golang](https://github.com/ravinsharma7/rust-plus-golang)
